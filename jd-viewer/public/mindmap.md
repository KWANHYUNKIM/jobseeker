## 모든 직군 공통 — 가산점·공부법
- 협업·커뮤니케이션 — 어떤 직군이든 1순위
  - 기술 글쓰기 (PR 설명, 설계 문서)
  - 코드 리뷰 주고받기 — 이유까지 설명
- 포트폴리오·아웃풋
  - GitHub 활동 (꾸준한 커밋 < 의미있는 프로젝트)
  - 기술 블로그 (배운 것 회고 + 깊은 글 1-2편)
  - 오픈소스 PR 1-2건 (Good first issue 부터)
- CS 기본기 — 시니어로 갈수록 결정적
  - 자료구조·알고리즘, 운영체제, 네트워크, DB 내부
- 영어
  - 문서 읽기 (공식 docs, RFC)
  - 외국계/원격 채용 시 결정적
- 사이드프로젝트 / 토이 → MVP 출시
  - '동작하는 무언가'를 만들어 사용자/매출을 가져본 경험
- 커뮤니티
  - 모임 발표, 컨퍼런스 라이트닝 토크
  - 멘토링 받기/하기
- 자격증 (가산점 채용 일부)
  - 클라우드 (AWS SAA/SAP, GCP PCA)
  - 정보처리기사 (공공/대기업 가산)
  - 보안 (CISSP, OSCP)
